# magnifying-glass

## 上手指南

  #### 下载  

  ```shell
npm i magnifying-glass
  
  ```

  #### 文件里面引入 

  ``` vue
  import MagnifyingGlass from 'magnifying-glass';

  ```


  #### 使用

  ``` vue
  
  <magnifying-glass :mianImages="图片地址"/>

  ```


  #### 图片样式可直接在class里设置，例如：

  ``` vue 
    <magnifying-glass :mianImages="图片地址" class="mianImages"/>

  ```

``` css
# 使用的是scss

  .mianImages {
    with: 400px;
    hight: 400px;
  }

```
